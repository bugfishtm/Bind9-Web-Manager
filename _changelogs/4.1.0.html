Version 4.1.0 Release

Description: Major UI Overhaul with expanded functionality, including a fully integrated DNS Server capable of standalone Domain and Record Management.

Key Features:

  • UI Overhaul: Completely redesigned interface for an improved user experience and more intuitive navigation.
  • Domain & Record Management: Full integrated management of DNS Domains and Records directly within the web interface.
  • Standalone DNS Server: Now capable of operating as a standalone DNS server without external dependencies.
  • And much more...

How to update 3.7.1 to 4.1.0:

  • Login to your web server.
  • Create a backup of Settings.php.
  • Remove settings.php and all existing DNSHTTP-related web files from the server (e.g. var/www/html/*).
  • Upload the new files from the _source folder.
  • Complete the installer at the website's root URL, then compare your configuration against the backed-up settings.php.
  • The installer will generate a new settings.php at _data/settings.php.
  • No database migrations or other major changes are required.

Backwards Compatibility:

  • Version 4.1.0 supports cross-version replication to and from version 3.x.x.

Please note that this release focuses on interface and architectural improvements. We remain committed to delivering a reliable Bind9 Web Manager experience and continue to be responsive to all support requests. There are no known bugs in this release.
