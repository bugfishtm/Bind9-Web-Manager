Version 4.1.1 Release

Description: Minor improvements following to previous major release and fixed a major replication error.

Key Features:

  • UI Improvements: Minor UI Improvements.
  • Stability: Minor Stability Improvements.
  • Major Issue: Fixed a major breaking replication issue on API requests with compression.

How to update 4.1.0 to 4.1.1:

  • Login to your web server.
  • Upload the new files from the _source folder. (Overwrite the old files, do not delete the old files.
  • No database migrations or other major changes are required.

Backwards Compatibility:

  • Version 4.1.1 supports cross-version replication to and from version 4.x.x/3.x.x.
