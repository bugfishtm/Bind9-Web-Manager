Version 3.7.3 Release

Description: This release of Bind9 Web Manager offers a stable and feature-rich solution for managing BIND9 DNS servers. While no new features have been added in this release, it comes with ongoing support and resolves all known bugs. We are committed to providing reliable assistance for your DNS management needs.

Key Features:

  • Slave Server Replication: Efficiently manage replication processes between Slave and Master Servers with real-time status updates.
  • Slave DNS Server Hybrid Support: Configure servers to serve as both Master and Slave Servers for streamlined DNS architecture management.
  • User Management: Create, edit, and delete users with granular permission assignment to tailor your workflow securely.
  • Rights Management: Precisely control user activities within designated operational areas, including "jailing" for domain restrictions.
  • IP Blacklisting: Control and mitigate security threats with comprehensive IP blacklisting, including the option to lift IP bans.
  • Replication Insights: Gain detailed insights into replication and domain information for informed decision-making.
  • Domains, Replications, and Conflicts: Easily manage replicated domains and address conflicts within the dedicated section.

Please note that while no new features have been added in this release, we remain dedicated to supporting your Bind9 Web Manager experience, and we are responsive to support requests. There are no known bugs in this release.
